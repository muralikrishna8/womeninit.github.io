First Edition :: Beginners' bootcamp :: DEV-QA

ThoughtWorks Hyderabad is glad to announce its beginners’ boot camp for female developers and QAs. This workshop is designed for all women who want to level up or brush up on their development or QA skills, but especially those who have taken a career break and want to return to the tech world. This 2-day event will run as parallel dev and QA tracks, filling in knowledge gaps and providing hands-on experience.

dev

  • Day 1: Sat 16th April
  • The current technology landscape
  • Analysis of problem statement
  • Basics: git, pair programming intro and machine setup
  • Hands-on programming
  • Review and wrap up
  • Day 2: Sun 17th April
  • Team practices
  • Pair programming
  • Intro to unit testing and TDD
  • Architecture of web applications (HTTP, MVC)
  • Executing a mini project
  • Review and wrap up

QA

  • Day 1:: Sat 16th April
  • Refresher: software testing concepts
  • Introduction to Java - basics, writing simple programs (hands-on)
  • Introduction to Java - understanding OOPs concepts, designing Java programs on IDE (hands-on)
  • Introduction to Selenium IDE (hands-on)
  • Day 2:: Sun 17th April
  • Introduction to Selenium WebDriver
  • Writing simple programs using Selenium Webdriver
  • Element location techniques (CSS and XPATH)
  • Automating a simple web page using Selenium Webdriver
